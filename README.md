# 🚀 Sistema Inteligente para Gestão de PMEs Baseado em LLMs

Um sistema web inteligente desenvolvido para otimizar a gestão de pequenas e médias empresas através de análise em tempo real com inteligência artificial.

## 📋 Visão Geral

O **PME Manager** é uma solução completa de gestão empresarial que combina funcionalidades tradicionais de gestão com insights inteligentes baseados em Large Language Models (LLMs). O sistema oferece uma interface moderna e intuitiva para gerenciar cronogramas de funcionários e agendamentos de clientes, enquanto utiliza IA para fornecer análises preditivas e recomendações estratégicas.

## ✨ Funcionalidades Principais

### 🕒 Cronograma de Funcionários
- Gestão completa de turnos e escalas
- Controle de presença e ausências
- Otimização automática de distribuição de pessoal
- Detecção de conflitos de horários
- Relatórios de produtividade por equipe

### 📅 Módulo de Agendamento
- Sistema de reservas para clientes
- Calendário integrado com múltiplas visualizações
- Gestão de conflitos de horários
- Notificações automáticas para clientes
- Histórico completo de agendamentos

### 🤖 Insights Inteligentes (LLM)
- Análise em tempo real dos dados operacionais
- Recomendações de otimização de recursos
- Previsões de demanda e ocupação
- Alertas automáticos para situações críticas
- Relatórios inteligentes personalizados

## 🚀 Como Executar o Projeto

### Pré-requisitos
- Navegador web moderno (Chrome, Firefox, Safari, Edge)
- Servidor web local (opcional para desenvolvimento)
- PostgreSQL (para funcionalidades completas)

### Instalação Local

1. **Clone o repositório**
   ```bash
   git clone [URL_DO_REPOSITORIO]
   cd TCC
   ```

2. **Abra o projeto**
   - Navegue até a pasta `html/`
   - Abra `dashboard.html` em seu navegador
   - Ou configure um servidor local simples

3. **Servidor Local Simples (Python)**
   ```bash
   # Python 3
   python -m http.server 8000
   
   # Python 2
   python -m SimpleHTTPServer 8000
   ```

4. **Acesse o sistema**
   ```
   http://localhost:8000/html/dashboard.html
   ```
